# 🏫 Excellence School Database Management System

A relational database project developed using **MySQL** as part of the **Database Design and SQL** course on **Satr Platform by Tuwaiq Academy**.

## 📖 Project Overview

This project simulates a centralized database for **Excellence Secondary School**. It manages information about students, teachers, and subjects while demonstrating the fundamentals of relational database design and SQL.

## ✨ Features

- Create a relational database
- Create and manage tables
- Insert sample data
- Update records
- Rename tables
- Sort data using ORDER BY
- Use aliases with AS
- Display table contents
- Apply SQL comments for documentation

## 🗂️ Database Tables

- Students
- Teachers
- Subjects (Renamed to Courses)

- SQL
